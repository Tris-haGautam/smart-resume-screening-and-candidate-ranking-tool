# Smart Resume Screening and Candidate Ranking Tool

## Overview
This project is an AI-powered Resume Screening Tool built using Python and Streamlit. It helps recruiters compare resumes with a job description and rank candidates based on their match score.

## Features
- Upload multiple resumes (PDF/DOCX)
- Upload Job Description (PDF/DOCX/TXT)
- Resume ranking based on TF-IDF and Cosine Similarity
- Candidate match score calculation
- AI-based candidate analysis
- Interactive Streamlit interface

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- PyPDF2
- python-docx

## Installation

```bash
pip install -r requirements.txt
streamlit run smart.py
